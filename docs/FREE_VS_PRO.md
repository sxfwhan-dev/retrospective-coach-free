# Free vs Pro Boundary

## Version Positioning

| Version | Positioning | Best For |
| --- | --- | --- |
| Free | Lightweight, usable, easy to share | Individuals, learners, lightweight retrospective scenarios |
| Pro | Systematic, deeper, easier to adopt | Creators, managers, team leads, consultants |
| Team | Team mechanisms and collaboration workflows | Business teams, project teams, management teams |
| Enterprise | Private deployment and custom delivery | Enterprise customers, training organizations, consulting firms |

## Feature Boundary

| Capability | Free | Pro |
| --- | --- | --- |
| General Retrospective Coach agent prompt | Included | Included |
| Natural-language retrospective report | Included | Included |
| Separation of facts, assumptions, and missing information | Included | Included |
| Basic output templates | Included | Included |
| Deep scenario templates | Not included | Included |
| Detailed workflow version | Not included | Included |
| Project, sales, interview, content, and team-specific templates | Not included | Included |
| High-quality case library | A few public examples | Included |
| Team retrospective meeting workflow | Not included | Included |
| Retrospective quality assessment sheet | Not included | Included |
| Coze / Dify / GPTs deployment packages | Not included | Included |
| Private deployment and industry customization | Not included | Optional |

## Acceptance Standards

| Version | Acceptance Standard |
| --- | --- |
| Free | Summarizes facts, identifies key issues, separates facts from assumptions, and gives 3-5 concrete improvement suggestions |
| Pro | Asks targeted follow-up questions, generates RCA, outputs prioritized action plans, includes owner / deadline / acceptance criteria, and turns lessons into team mechanisms |

## Capability Depth Boundary

The Free version can provide:

- Basic fact reconstruction.
- Goal-vs-result gap identification.
- Key issue and risk-chain identification.
- Lesson extraction.
- Lightweight action suggestions.

The Pro version can further provide:

- Multi-turn coaching questions.
- RCA, 5 Whys, fishbone diagrams, or timeline analysis.
- Prioritized action plans.
- Owner / deadline / acceptance criteria.
- Monitoring thresholds, alert rules, and release gates.
- Team mechanism design and retrospective quality assessment.

## Suggested Public Wording

You can describe the boundary like this in README or product pages:

```text
Retrospective Coach Free is permanently available for personal learning, lightweight retrospectives, and community adaptation.

If you need a more complete retrospective system, including multi-scenario templates, detailed workflows, case libraries, team mechanisms, and agent deployment packages, consider the Pro version.
```

## What the Free Version Should Not Include

- Complete detailed workflow assets.
- Pro prompt assets.
- Paid case libraries.
- Team training slides.
- Consulting delivery templates.
- Private deployment configuration.
- Customer project materials.

## Pro Value Proposition

- More complete: covers more scenarios than the general version.
- More stable: includes deeper workflows, check points, and quality controls.
- More time-saving: ready to deploy, reuse, and operate with teams.
- More deliverable: supports consulting, training, team meetings, and organizational learning.

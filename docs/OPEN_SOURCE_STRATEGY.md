# Open Source Strategy

This project uses an open-source adoption strategy:

The Free version is published on GitHub to spread the idea, let people try it, build trust, and collect community feedback.

The Pro version stays paid and closed-source. It carries the more complete system, deeper scenario templates, more mature deployment packages, and team implementation materials.

## Core Principles

1. The Free version must be genuinely useful, not just a teaser.
2. The Free version should solve general lightweight retrospective needs, not deep team implementation.
3. The Pro version should not merely be "a longer prompt"; it should provide a more complete, more stable, and more time-saving solution.
4. The repository should clearly explain the Free/Pro boundary.
5. Community adaptation is allowed, while the Pro product and commercial service space remain protected.

## License Choice

This repository uses CC BY 4.0.

Why:

- It fits prompts, templates, documentation, and examples.
- It allows sharing and commercial use, which helps distribution.
- It requires attribution, which preserves source visibility.

If code is added later, consider:

- Code: MIT or Apache-2.0
- Prompts / docs / examples: CC BY 4.0

## Conversion Path

Keep a light Pro conversion entry in the README:

```text
For the Pro version, scenario templates, team deployment packages, or enterprise customization, see the Pro entry announced in the README.
```

If there is no purchase link yet, write:

```text
The Pro version is in preparation. Please open an issue to share which retrospective scenarios you need most.
```

## Content Release Rhythm

Stage 1: publish the core Free package:

1. Free prompt
2. Usage guide
3. Public examples
4. Free/Pro boundary
5. Launch checklist or roadmap

Stage 2: add based on feedback:

1. More public cases
2. Usage notes for different AI tools
3. FAQ
4. Community-contributed cases

Stage 3: launch Pro:

1. Pro product page
2. Purchase or subscription entry
3. Sample chapter or preview
4. User cases and testimonials

# Contributing

Thank you for helping improve Retrospective Coach Free.

## What You Can Contribute

- New retrospective examples
- Clearer prompt wording
- More concrete action templates
- Usage notes for different AI tools
- Better alternatives to vague advice
- Typos, formatting fixes, and documentation improvements

## Contribution Principles

Please make sure your contribution follows these principles:

1. Keep the output natural-language first. Do not add JSON output requirements.
2. Do not encourage invented facts, numbers, people, budgets, competitors, or historical cases.
3. Assumptions must be labeled as assumptions or hypotheses to verify.
4. Improvement actions should include a first step, timing, or check signal.
5. Avoid vague advice such as "communicate better", "improve execution", or "close the loop" unless it is translated into concrete behavior.
6. Do not submit Pro, paid, private, or unpublished commercial materials.

## Issue and Pull Request Suggestions

When opening an issue, please include:

- The retrospective scenario you encountered
- What did not work well in the current prompt
- What you would like to improve
- Whether you can share a public example

For pull requests, keep changes small and clear. If you want to make a large rewrite, please open an issue first.
